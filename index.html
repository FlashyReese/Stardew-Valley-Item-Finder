Stardew Valley Price Calculator

Instructions

The save file is of the form Name_123456789. It can be found in the following folders.

  • Windows: %appdata%\StardewValley\Saves
  • Linux and macOS: ~/.config/StardewValley/Saves

You can sort the table by clicking on the headers, and you can download the table as a CSV file to use in Excel.

About

Currently, this tool does not account for any professions, which can make some of the prices inaccurate at higher skill levels.

This tool runs entirely inside the browser. Your save file is never sent across the network. All of the data is extracted using xslt if that sort of thing interests you.

Source repository located at https://github.com/gucci-on-fleek/Stardew-Valley-Price-Calculator. Inspired by the Stardew Fair Helper
